# CUUC Vision Statement Project

Working files for the Committee on Ministry's effort to update CUUC's vision statement and bring a new statement to a congregational vote in late 2026.

## Where things are

| Folder | What's in it |
|---|---|
| [`01_process/`](01_process/) | Charter, timeline, decisions log — the working spec for *how* we're running this. |
| [`02_reference/`](02_reference/) | Background reading: what a vision statement is, current CUUC statements, UU example statements, UUA resources, prior visioning processes, candidate questions, COM's SPARC Q5 input, and a meeting-specific UUA synthesis. |
| [`03_meetings/`](03_meetings/) | Per-meeting agendas, summaries, homework. One folder per meeting date. |
| [`04_workshop_design/`](04_workshop_design/) | The strawman input service (*Dreaming Forward Together*) for Oct 18, 2026. |
| [`05_drafting/`](05_drafting/) | (Empty for now.) Will hold candidate vision statements after the Oct 18 input service. |
| [`06_correspondence/`](06_correspondence/) | Email threads (`.eml` files) — assumption-check with minister/Board, post-meeting summaries. |
| [`_originals/`](_originals/) | Original `.docx` files preserved as-downloaded. The cleaned-up markdown versions live elsewhere in this tree. |
